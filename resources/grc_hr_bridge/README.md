# grc_hr_bridge Resources

This folder is reserved for non-installable bridge assets.

Use it for:

- the source markdown form corpus under `forms/`
- generated PDFs
- checklist exports
- role template output samples
- interview rubric artifacts
- document pack evidence

Do not include these assets inside the Odoo module zip.

The markdown forms are the human-readable source inventory for the HR bridge. Generated PDFs and evidence artifacts should live here, not in the installable addon directory.
