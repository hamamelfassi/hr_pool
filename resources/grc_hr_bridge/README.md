# Recruitment Governance Bridge Resources

This folder is reserved for non-installable bridge assets.

Use it for:

- the source markdown form corpus under `forms/`
- generated PDFs
- checklist exports
- role template output samples
- interview rubric artifacts
- document pack evidence

Do not include these assets inside the Odoo module zip.

The markdown forms are the human-readable source inventory for the recruitment bridge. Generated PDFs and evidence artifacts should live here, not in the installable addon directory.

Note:

- this directory name is still the legacy filesystem path while the scaffold is being renamed
- the intended runtime module name is `grc_recruitment_bridge`
